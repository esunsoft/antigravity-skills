# Skill 评分标准

## 8 大工程设计原则

| # | 原则 | 含义 |
|---|------|------|
| P1 | 单一职责 | 每个 Skill 只承担一类具体任务 |
| P2 | 稳定接口 | 结构化的输入/输出接口 |
| P3 | 可选择调用 | 精确的触发条件和 description |
| P4 | 确定性与可验证性 | 执行结果可预测、可验证 |
| P5 | 显式失败与恢复 | 允许显式失败并支持恢复 |
| P6 | 模型解耦 | 与特定模型和 Prompt 解耦 |
| P7 | 可复用、可组合、可演进 | 独立测试、迭代和演进 |
| P8 | 安全性 | 不含恶意行为，不泄露数据 |

## 维度权重

| 维度 | 权重 |
|------|------|
| Structure | 15% |
| References | 15% |
| Triggers | 10% |
| Design | 20% |
| Security | 25% |

> 安全 Blocker 会使整个 Skill 评分降至 F 级（≤ 39）。
